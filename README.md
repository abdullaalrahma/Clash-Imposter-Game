
# Clash Imposter 🎮

## Overview
Clash Imposter is a browser-based social deduction game inspired by games like Mafia and Among Us, but themed around characters from Clash Royale. The game focuses on strategy, communication, and logical thinking, where players must identify the imposter among them.

## Features
- 🧠 Social deduction gameplay (Mafia-style mechanics)
- 👥 Multiple player roles (Imposter vs Crew)
- 🗳️ Voting and elimination system
- 🎯 Strategy-based decision making
- 🌐 Runs directly in the browser (HTML-based)

## How to Play
1. Open the game in your browser.
2. Each player is assigned a hidden role:
   - *Crew*: Work together to find the imposter.
   - *Imposter*: Blend in and eliminate other players without being detected.
3. Players discuss and vote to eliminate suspicious players.
4. The game ends when:
   - The imposter is eliminated (Crew wins)
   - The imposter outnumbers or eliminates the crew (Imposter wins)
